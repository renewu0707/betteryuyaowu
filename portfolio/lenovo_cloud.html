INTRODUCTION

Lenovo Cloud OS Network, we use COSN as the name of this project, is a IaaS public cloud service portal constructed on Lenovo existed hardware environment, including functions of Tenant Application and Log-in, VMs Service Application and Modification, VMs Management, etc. (VM is short for Virtual Machine.). The portal we build for Lenovo will help him to become a cloud service provide, and offer enterprise tenants overall basic VM service.

The portal has three sub-portals according to different roles: Cloud Computing Portal for Tenant Admin, Self-service Portal for Tenant Admin & Tenant Users, and Cloud Service Management Portal for Service Provider.
