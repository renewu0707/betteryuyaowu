INTRODUCTION

Lenovo Cowork platform is implemented based on Microsoft SharePoint 2013, and interaged with Microsoft Lync and Exchange. The platform aims at providing self-help site creating service, in order to achieve more convenient work mode, such as sharing, collaboration, and communicating, the supportive site types include Personal Site, Department Site, Team Site, Promotion Site, Project Site and Discussion Site.

WHAT'S MY ROLE ?

Besides the below design work, I also assisted in web development and testing, prepared Using Handbooks for our customers, and constructed Help Site, etc.

I designed the Team Site Template. I analyzed frequency of use for different functions, select 5 function parts to be shown in first level of web page, set up site navigation bar, and design the title and cue words for each part.

5 main function parts are:
(from top to bottom, left to right)

Team Photos
Activity Information - it is actually a document library
Team Announcement
Team Tasks
Newsfeed

I also designed the Project Site Template. This template refers to the top 9 knowledge areas in modern project management.

6 main function parts are:
(from top to bottom, left to right)

Project Plans
Project Risks
Project Announcements
Project Internal Document
Project Pubblished Document
Newsfeed

What's more, I designed the Department Site Template. Since the department site would be used for the whole department, it had to be special. The structure was different from the previous two sites.
